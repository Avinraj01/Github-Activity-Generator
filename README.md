# 🧠 GitHub Activity Generator

> 🚀 Generate custom GitHub contribution graph using Python

---

## 📌 Features

* Generate backdated commits
* Custom date range support
* Random commit intensity
* Fully automated

---

## ⚙️ Tech Stack

* Python
* Git
* GitHub

---

## 🚀 How to Use

1. Clone repo
2. Run:
   python3 activity.py

---

## 📊 Output

Generates a green contribution graph on your GitHub profile.
